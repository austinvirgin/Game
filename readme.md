# Project Title (Update)

This is a game demonstration with a bot that also plays pong.


## Instructions for Build and Use

Steps to build and/or run the software:

1. Download the code from vscode and create a virtual environment

Instructions for using the software:

1. Start the virtual environment and download all of the things that are imported
2. Run testbot.py and watch the bots play against each other

## Development Environment 

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Gymnasium
* Stable Baselins 3
* Numpy
* Arcade
